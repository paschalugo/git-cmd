•Branch (B)

A branch is just a separate line of development.
Its a way to work on muliple features in a project without it affecting the main codebase.

•git Initializing (init)

This is used in creating new git repository.

•git status

Is used to check the git status of your software

•Commit

This is a command that helps you to save changes to your repository
It is also known as checkpoint, just like in video games
when you want to exit the game you save it so when next you want to play the game it start from where you saved it that is how the (commit) command works

•git push

use this command when trying or want to move your works or folder to a cloud such as github etc.

•git configuration (config)

This is used to manage your git settings for example
git config --global user.name "your name"

•Record of operations (git log)

this is a command used to show all the commits made to a repository.

•git ignore

this is used to ignore some files you dont want to be moving them every where.

•LS

it is used to display or list the files and directories in a directory

•git clone (url)

It is used to clone a git repository from a specified url.

•git pull

it is used to pull changes fromr your remote repository and merges them into local repository.

•git fetch

it feches changes from a remote repository.

•git reset --hard

it reset the repository to the last commit and discharge all changes

•git checkout

it is used to switch to different branch

•git merge

it joins changes from another branch into the current branch

•git reset --soft

Reset the repository to the last commit but it preserve changes.

•git tag

Is used to create a new tag

•git branch --list

This command is used to list all local branches

•git add

it is used to stage changes in your working dictionary preparing them to be commited

•git clean -f

it is used to delete all untracked files

•git add -A

this command is used to stage all changes including new,modified and deleted files.

•git add -U

It is used to stage only modified and deleted files.
